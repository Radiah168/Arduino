# 4-Bit Binary LED Counter

## Description
This project demonstrates a 4-bit binary counter using Arduino and four LEDs.
The LEDs display binary numbers from 0000 to 1111 by turning ON and OFF in sequence.

## Components Used
- Arduino Board
- 4 × LEDs
- 100Ω and 1000Ω resistors
- Jumper wires
- Breadboard

## Working Principle
Each LED represents a binary bit. The Arduino controls the LEDs through digital pins and changes their states after a fixed delay.
The counter starts from 0000 and increases up to 1111, then repeats continuously.

## Pin Configuration
- LED 1 → Pin 13
- LED 2 → Pin 9
- LED 3 → Pin 7
- LED 4 → Pin 4

## Delay Time
750 ms
