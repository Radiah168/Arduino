# LED Blink Arduino Project

## Description
This project demonstrates an Arduino LED blinking program using an LED connected to pin 13 with a 100Ω resistor for current protection.

The LED blinks in three different speed levels:
- Fast blinking (50 ms delay)
- Medium blinking (700 ms delay)
- Slow blinking (1500 ms delay)

This project helps to understand basic Arduino programming, digital output control, and delay functions.

## Components Used
- Arduino Board
- LED
- 100Ω Resistor

## Working Principle
The Arduino sets pin 13 as an output pin. The LED is turned ON and OFF using the `digitalWrite()` function. Different `delay()` values are used to control the blinking speed.
